Searching & Filtering

The search box at the top of the collection allows you to quickly find Squishmallows by various criteria:

  • By Name: Type any part of a Squishmallow's name (e.g., "cam", "bunny")
  • By Number: Search by Squishmallow number (e.g., "001", "1234")
  • By Size: Filter by size category (e.g., "8 inch", "12 inch")
  • By Squad: Find all members of a squad (e.g., "farm squad", "ocean squad")
Pro Tip:

The search is case-insensitive and matches partial text. Try searching for colors, themes, or any text that appears in the Squishmallow details!

Tracking Your Collection

Squishmallowdex includes two ways to track your collection, with all data stored locally on your device:

‚ù§Ô∏è Favorites

Click the heart icon next to any Squishmallow to mark it as a favorite. Your favorites are highlighted and can be filtered using the "Show Only Favorites" button at the top of the page.

‚úì Ownership Tracking

Check the box next to any Squishmallow you own. You can filter to show only owned Squishmallows using the "Show Only Owned" button. This is perfect for tracking which ones you still need to collect!

Privacy First:

All your favorites and ownership data is stored in your browser's local storage. It never leaves your device - no accounts, no tracking, no servers!

Sorting

Click any column header to sort your collection by that field:

  • Number: Sort by Squishmallow ID number
  • Name: Alphabetical order by name
  • Size: Group by size categories
  • Squad: Organize by squad membership

Click the same header again to reverse the sort order (ascending ‚Üî descending).

Offline Usage

Squishmallowdex is a Progressive Web App (PWA) that works completely offline after the first load. This means:

  • View your entire collection without internet
  • Search, sort, and filter all work offline
  • Your favorites and ownership data is always available
  • No data usage - perfect for mobile devices!
How it Works:

The collection page includes all images and data embedded directly in the HTML file. Once loaded, everything is cached in your browser for instant offline access.

Favorites/Ownership Not Saving

If your favorites or ownership tracking isn't persisting between sessions, your browser may have localStorage disabled. Check your browser's privacy settings and ensure local storage is enabled.

Search Not Working

Make sure JavaScript is enabled in your browser. Squishmallowdex requires JavaScript for search, sort, and filtering features.

File Too Large

The full collection HTML file can be quite large (35+ MB) due to embedded images. This is normal! Modern browsers can handle this size easily. If you experience performance issues, try using a different browser or viewing on a device with more memory.
