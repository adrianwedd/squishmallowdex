About Squishmallowdex

Squishmallowdex is your Pokédex for Squishmallows - a cozy, colorful collector's guide that helps you browse, search, and track your Squishmallow collection. Like a Pokédex helps trainers catalog Pokémon, Squishmallowdex helps collectors organize and explore over 3000 adorable Squishmallows!

This project gathers information from the Squishmallows Wiki and transforms it into a magical, searchable collection you can open anytime - even without an internet connection.

Educational Mission

Squishmallowdex is designed with young collectors and learners in mind. It's more than just a collection tracker - it's an educational tool that teaches:

  • Organization Skills: Learn to catalog and organize collections systematically
  • Research & Discovery: Explore different squads, sizes, and characteristics
  • Technology Literacy: Experience how web applications work offline
  • Privacy & Safety: Understand local data storage and why we don't track users

Made with love for young collectors who want to learn along the way!

Technology Stack

Squishmallowdex is built using modern web technologies while keeping everything simple, accessible, and privacy-focused:

Python Data collection and HTML generation
Vanilla JS No frameworks, just pure JavaScript
PWA Progressive Web App for offline use
LocalStorage Your data stays on your device

Why These Choices?

  • No dependencies: The collection works standalone - just open the HTML file
  • Privacy-first: All data stored locally, no tracking, no analytics, no ads
  • Offline-capable: Add to home screen and use anywhere, anytime
  • Educational: Simple codebase that learners can explore and understand

Features

  • 3000+ Squishmallows: Comprehensive database with images and details
  • Powerful Search: Find by name, number, size, or squad
  • Collection Tracking: Mark your favorites with hearts and track ownership
  • Sort & Filter: Organize by name, number, size, or squad
  • Responsive Design: Works beautifully on desktop, tablet, and mobile
  • Kid-Friendly UI: Colorful, intuitive interface for all ages
  • No Internet Required: After initial load, works completely offline

Open Source & Community

Squishmallowdex is an open-source, fan-made project. The code is available on GitHub for anyone to explore, learn from, or contribute to.
